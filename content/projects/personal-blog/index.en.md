# Personal Academic Blog

## Project Overview

A purely static personal academic blog based on GitHub Pages, built with HTML/CSS/JS without any build tools or frameworks.

## Features

- **Bilingual Support**: All UI text and content supports Chinese/English switching
- **Markdown Rendering**: Course notes and project docs written in Markdown, auto-rendered to HTML
- **Course Management**: Multi-course, multi-chapter note organization with sidebar navigation
- **Project Showcase**: Project card display + detail page rendering
- **Responsive Design**: Adapts to desktop, tablet, and mobile
- **Web Components**: Custom elements for cross-page nav/footer reuse

## Tech Stack

| Technology | Purpose |
|------------|---------|
| HTML5 | Page structure |
| CSS3 | Styling & responsive layout |
| JavaScript (ES6+) | Interactions, i18n, dynamic rendering |
| marked.js | Markdown parsing & rendering |
| Font Awesome | Icon library |
| GitHub Pages | Hosting & deployment |
