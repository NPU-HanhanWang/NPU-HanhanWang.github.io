# 第三章：图像分割

## 3.1 图像分割概述

图像分割的目标是将图像划分为若干具有语义意义的区域。是目标检测、场景理解等任务的基础。

主要方法分类：
- **传统方法**：基于阈值、边缘、区域
- **机器学习方法**：聚类、图割
- **深度学习方法**：FCN、U-Net、Mask R-CNN

## 3.2 阈值分割

### 大津法（Otsu's Method）

Otsu 算法自动寻找最佳阈值，使得类间方差最大。

```python
import cv2

gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
_, binary = cv2.threshold(gray, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
```

## 3.3 基于深度学习的分割

U-Net 是医学图像分割中最经典的架构，采用编码器-解码器结构：

- **编码器（下采样路径）**：提取多尺度特征
- **解码器（上采样路径）**：恢复空间分辨率
- **跳跃连接**：融合不同尺度的特征

## 总结

图像分割是计算机视觉的基础任务之一。传统方法适合简单场景，深度学习方法在复杂场景中表现更优。
