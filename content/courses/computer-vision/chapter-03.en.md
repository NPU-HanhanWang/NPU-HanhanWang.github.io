# Chapter 3: Image Segmentation

## 3.1 Overview

The goal of image segmentation is to partition an image into semantically meaningful regions. It serves as a foundation for object detection, scene understanding, and other tasks.

## 3.2 Threshold-Based Segmentation

Otsu's method automatically finds the optimal threshold by maximizing between-class variance.

## 3.3 Deep Learning-Based Segmentation

U-Net is the most classic architecture for medical image segmentation, using an encoder-decoder structure with skip connections.

## Summary

Image segmentation is a fundamental task in computer vision. Traditional methods work well for simple scenes, while deep learning methods excel in complex scenarios.
