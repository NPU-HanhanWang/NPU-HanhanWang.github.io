# Computer Vision

## Course Introduction

This course systematically introduces core theories and algorithms in computer vision, covering the complete knowledge system from traditional image processing to modern deep learning methods.

## Learning Objectives

- Master basic concepts and operations of image processing
- Understand core algorithms of feature extraction and matching
- Be familiar with classical and deep learning methods for image segmentation
- Master mainstream frameworks for object detection
- Be able to implement computer vision algorithms using PyTorch and OpenCV

## Reference Books

1. *Computer Vision: Algorithms and Applications* — Richard Szeliski
2. *Deep Learning for Vision Systems* — Mohamed Elgendy
3. *Digital Image Processing* — Gonzalez & Woods
