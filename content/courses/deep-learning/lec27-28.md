# 一、Hopfield 网络：从循环到联想记忆

## 1.1 核心思想：物理学启发的计算模型

Hopfield 网络是一种**全连接、循环、对称权重的二值神经网络**。它受到**自旋玻璃 (Spin Glass)** 模型的启发，将网络的状态与物理系统的**能量 (Energy)** 概念联系起来。

-   **2024 年诺贝尔物理学奖**：John Hopfield 因其在利用物理学工具构建现代机器学习基础方面的工作而获奖，特别是他创造的**联想记忆 (Associative Memory)** 模型。

    ```
        Hopfield 网络结构 (N=4)：

              y1 o <---w12---> o y2
                | \           / |
                |   \        /   |
                |     \    /     |
               w13     w24       w23
                |       /  \      |
                |     /     \     |
                |   /        \    |
              y3 o <---w34---> o y4

        特点:
        - 权值对称: w_ij = w_ji
        - 无自环: w_ii = 0
        - 每个神经元输出 y_i ∈ {+1, -1}
    ```

## 1.2 更新规则与能量函数

-   **更新规则 (同步或异步)**：
    <br>
    $$ y_i = \text{sgn} \left( \sum_{j \neq i} w_{ji} y_j + b_i \right) $$
    <br>
    神经元根据其**局部场 (Local Field)** 决定是否翻转，以与局部场对齐。

-   **能量函数 (Energy Function)**：
    <br>
    $$ E = -\frac{1}{2} \sum_{i,j} w_{ij} y_i y_j - \sum_i b_i y_i = -\frac{1}{2} \mathbf{y}^T \mathbf{W} \mathbf{y} - \mathbf{b}^T \mathbf{y} $$
    <br>
-   **收敛性证明**：每次翻转，能量 $E$ 都会严格减小，且 $E$ 有下界。因此，网络必然在有限步内收敛到一个局部能量最小值。

## 1.3 联想记忆：从受损输入中恢复完整模式

-   **工作原理**：能量景观中的每一个**局部最小值**都对应一个"记忆"模式。当网络被初始化为一个受损或部分模式时，它将通过能量下降过程，演化到该"记忆"所在的"能量井"底部，从而完成模式恢复。
-   **内容可寻址内存 (Content-Addressable Memory)**：你不需要知道记忆的"地址"，只需提供一个与目标记忆"相似"的线索，网络就能通过能量最小化找到最接近的记忆。

    ```
        联想记忆：从噪声中恢复

        +--------+     +--------+     +--------+     +--------+
        |        |     |        |     |        |     |        |
        |   #    |     |   #    |     |   #    |     |   #    |
        |  ###   | --> |  ###   | --> |  ###   | --> |  ###   |
        |   #    |     |   #    |     |   #    |     |   #    |
        |        |     |        |     |        |     |        |
        +--------+     +--------+     +--------+     +--------+
        受损输入        迭代 1          迭代 2         完整记忆
        (噪声版"十"字)   (部分恢复)      (接近恢复)      (完美恢复)
    ```

## 1.4 赫布学习：如何"编程"记忆

-   **赫布规则 (Hebbian Learning)**："Fire together, wire together"。

-   **存储单个模式 $y^p$**：
    <br>
    $$ w_{ij} = y_i^p y_j^p $$
    <br>
-   **存储多个模式 $\{y^1, y^2, ..., y^K\}$**：
    <br>
    $$ w_{ij} = \frac{1}{N} \sum_{p=1}^{K} y_i^p y_j^p $$
    <br>
    (通常 $w_{ii} = 0$，且忽略偏置项)

-   **存储容量**：
    -   $N$ 个神经元的网络，使用赫布学习，最多可存储约 $0.14N$ 个**随机**模式。
    -   **容量限制**：来源于模式之间的串扰，当 $K$ 超过 $0.14N$ 时，串扰噪声会导致回忆失败或产生虚假记忆 (Spurious Memories)。
    -   **最大值**：理论上，通过更精细的优化（而非简单赫布规则），网络可以存储 $O(N)$ 个模式，但需要付出计算代价。

<details>
<summary><b>📊 为什么容量只有 0.14N？</b></summary>

对于 $N$ 个比特、$K$ 个随机存储的模式，对于一个神经元 $i$ 的输入信号，其"信号"部分（来自模式自身）为 $1$，而"噪声"部分（来自其他 $K-1$ 个模式）的方差为 $(K-1)/N$。随着 $K$ 增加，信号-噪声比降低。当 $K > 0.14N$ 时，噪声变得足够大，足以翻转神经元的正确状态，导致该模式不再是一个稳定的固定点。$0.14$ 这个数字源于正态分布假设下，噪声小于 -1（导致翻转）的概率小于 $0.4\%$ 的条件，即 $\sqrt{(K-1)/N} < 1 \implies K < 0.14N$。
</details>

## 1.5 霍普菲尔德网络的局限性

1.  **容量有限**：最大存储模式数 O(N)，限制其作为大容量内存的实用性。
2.  **虚假记忆**：产生大量非目标存储的局部最小值，消耗存储容量。
3.  **非确定性**：不同初始化可能导致不同记忆（吸引子），难以用于分类。
4.  **联想能力弱**：难以执行复杂的联想或推理任务。

# 二、玻尔兹曼机 (Boltzmann Machine)

## 2.1 从确定性到随机性：引入温度

玻尔兹曼机是 Hopfield 网络的**随机、概率性**扩展。它引入了"**温度 (Temperature)**"概念。

-   **随机更新规则**：神经元以一定的概率取 $+1$ 或 $0$ (或 $-1$)，概率由其**局部场**和**温度** $T$ 决定。
    <br>
    $$ P(s_i = 1) = \frac{1}{1 + \exp\left(-\frac{1}{T} \sum_j w_{ij} s_j\right)} $$
    <br>
-   **热平衡 (Thermal Equilibrium)**：通过**吉布斯采样 (Gibbs Sampling)**，网络最终会达到一个平衡分布，称为**玻尔兹曼分布 (Boltzmann Distribution)**：
    <br>
    $$ P(S) = \frac{1}{Z} \exp\left(-\frac{E(S)}{T}\right) $$
    <br>
    其中 $Z$ 是配分函数，$E(S) = -\sum_{i<j} w_{ij} s_i s_j$。

## 2.2 玻尔兹曼机作为生成模型

一个训练好的玻尔兹曼机定义了一个训练数据分布 $P_{\text{data}}(V)$ 上的概率模型。

-   **可见神经元 (Visible Units, V)**：对应观测数据。
-   **隐藏神经元 (Hidden Units, H)**：增加网络容量，学习数据中的潜在结构。
-   **网络分布**：
    <br>
    $$ P(V) = \sum_{H} P(V, H) $$
    <br>
-   **最大似然学习**：目标是调整权重 $w_{ij}$ 以最大化可见单元 $V$ 在训练数据上的对数似然。

## 2.3 最大似然学习的梯度

对可见单元对数似然的梯度进行推导，可以得到一个简单而优雅的**学习规则**：

$$
\frac{\partial \log P(V)}{\partial w_{ij}} = \langle s_i s_j \rangle_{\text{data}} - \langle s_i s_j \rangle_{\text{model}}
$$

-   $\langle s_i s_j \rangle_{\text{data}}$：在**数据分布**下 $s_i$ 和 $s_j$ 的期望协方差（即**夹钳相位**，Clamped Phase）。对于 $V$ 固定，$H$ 通过采样得到。
-   $\langle s_i s_j \rangle_{\text{model}}$：在**模型分布**下 $s_i$ 和 $s_j$ 的期望协方差（即**自由相位**，Free Phase）。所有 $V$ 和 $H$ 都自由地依模型分布演化。
-   **直观理解**：学习规则试图增加数据中同时出现的特征对之间的权重，同时减少模型自身幻想出来的特征对之间的权重。

## 2.4 受限玻尔兹曼机 (RBM)

由于完整玻尔兹曼机的采样过程非常慢，计算 $\langle s_i s_j \rangle_{\text{model}}$ 极其困难。**受限玻尔兹曼机 (Restricted Boltzmann Machine)** 通过限制网络结构来克服这一难题。

-   **核心约束**：**可见层单元之间无连接，隐藏层单元之间也无连接**。所有连接必须跨层，形成一个二分图。

    ```
        受限玻尔兹曼机 (RBM) 结构：

                h1      h2      h3      h4   ...   h_M (隐藏层)
                 o       o       o       o         o
                 | \   / | \   / | \   / |         |
                 |  \ /  |  \ /  |  \ /  |         |
                 |  / \  |  / \  |  / \  |         |
                 | /   \ | /   \ | /   \ |         |
                 o       o       o       o         o
                v1      v2      v3      v4   ...   v_N (可见层)

        特点: 层内无连接 (v1 与 v2 之间无连接，h1 与 h2 之间无连接)
    ```

-   **条件独立性的好处**：
    1.  在夹钳相位，给定可见单元 $V$，隐藏单元 $H$ 之间**条件独立**，可以高效并行采样。
    2.  在自由相位，给定隐藏单元 $H$，可见单元 $V$ 之间也**条件独立**，可以并行采样。
    3.  这大大加快了采样过程。

-   **对比散度 (Contrastive Divergence, CD)**：
    Hinton 提出，用于估计梯度的近似方法，**只需执行一步吉布斯采样**：
    <br>
    $$ \frac{\partial \log P(V)}{\partial w_{ij}} \approx \langle v_i h_j \rangle^0 - \langle v_i h_j \rangle^1 $$
    <br>
    -   $\langle \cdot \rangle^0$：用数据初始化 $V$，采样一次 $H$ 来计算。
    -   $\langle \cdot \rangle^1$：从 $\langle \cdot \rangle^0$ 的 $H$ 重构一次 $V'$，再采样一次 $H'$ 来计算。
    -   CD 训练速度快，是 RBM 的主流训练算法，虽然它是梯度的一个有偏估计，但在实践中非常有效。

# 三、总结与关联

| 模型                     | 神经元               | 连接                | 更新规则               | 学习算法                  | 主要用途                              |
| :----------------------- | :------------------- | :------------------ | :--------------------- | :------------------------ | :------------------------------------ |
| **Hopfield 网络**        | 确定性 ($\pm 1$)     | 全连接，对称        | 能量下降，确定性翻转   | 赫布规则 / 优化           | 联想记忆，模式补全                    |
| **玻尔兹曼机 (BM)**      | 随机 ($0/1$, $-1/1$) | 全连接，对称        | 吉布斯采样，概率性翻转 | 最大似然，需大量采样 (慢) | 生成模型，概率建模                    |
| **受限玻尔兹曼机 (RBM)** | 随机 ($0/1$, $-1/1$) | 二分图 (层内无连接) | 吉布斯采样，条件独立   | 对比散度 (CD) (快)        | 特征提取，深度信念网络 (DBN) 的预训练 |

### 关键联系

1.  **能量景观**：Hopfield 网络是 RBM 在温度 $T \to 0$ 时的特例，此时概率性采样退化为确定性"翻转至能量下降方向"。
2.  **最大似然与能量最小化**：BM 的最大似然学习，等价于在数据分布上降低能量（压低 $\langle E \rangle_{\text{data}}$），在模型分布上抬升能量（抬高 $\langle E \rangle_{\text{model}}$）。
3.  **隐藏单元的意义**：隐藏单元被视为学习到的潜在特征或"构图基元"，这与**自编码器 (Autoencoder)** 中的**潜在编码 (Latent Code)** 概念有异曲同工之妙。训练 RBM 可以看作是学习一个**带能量约束的、概率性的、生成式的自编码器**。
4.  **深度信念网络 (Deep Belief Networks, DBN)**：通过堆叠 RBM 形成深层生成模型。每个 RBM 层的输出作为下一层 RBM 的输入，逐层预训练后进行微调，曾是深度学习历史中重要的里程碑。
