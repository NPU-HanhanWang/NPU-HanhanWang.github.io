# 一、从判别到生成：问题的转变

## 1.1 分类网络的本质：后验概率估计

-   **核心观点**：一个训练好的分类神经网络，其输出（经过 Softmax）实际上是对**后验概率 $P(\text{class} \mid X)$** 的估计。
-   **对于线性不可分数据**：网络的前 $L-1$ 层可以被看作一个**特征提取器 (Feature Extractor)**，它将原始输入 $X$ 映射到一个新的特征空间 $h = f(X)$，在这个空间中，数据变得**线性可分**或**近似线性可分**。
-   **最后一层 (Softmax)**：则是在这个新特征空间上执行线性分类，计算 $P(y \mid h)$。

    ```
        分类网络作为概率模型：

        输入 X (如人脸像素)  -->  [特征提取网络] --> 特征 h (如眼睛大小、间距)
              |                       (非线性变换)          |
              |                                            v
              |                                   [线性分类器 (Softmax)]
              |                                            |
              |                                            v
              +--------------------------------------> P(类别 | X)

        * 特征提取：将原始像素空间扭曲成一个"线性可分"的流形。
        * 概率输出：网络输出的不是"这是猫"的决定，而是"这是猫的概率"。
    ```

## 1.2 自编码器 (Autoencoder, AE)

-   **目标**：学习一个恒等映射 $f(X) \approx X$，但中间有一个"信息瓶颈"。
-   **结构**：
    1.  **编码器 (Encoder)**：$Z = E(X; \theta_e)$，将高维输入压缩成低维的**潜在编码 (Latent Code)** $Z$。
    2.  **解码器 (Decoder)**：$\hat{X} = D(Z; \theta_d)$，从潜在编码 $Z$ 重构出输入 $\hat{X}$。
-   **损失函数**：最小化重构误差，通常使用 MSE。
    <br>
    $$ L_{\text{AE}} = \|X - D(E(X))\|^2 $$
    <br>

-   **线性自编码器与 PCA**：
    -   如果编码器和解码器都使用**线性激活函数**，自编码器学习到的权重矩阵会与**主成分分析 (PCA)** 的结果等价。编码器输出的是输入在主成分方向上的投影，解码器则是从主成分重建。

-   **非线性自编码器与流形学习**：
    -   当编码器和解码器使用**非线性激活函数**（如 ReLU, Sigmoid）时，网络可以学习数据背后的**非线性流形 (Nonlinear Manifold)**。
    -   **潜在空间的解耦 (Disentanglement)**：理想情况下，$Z$ 的不同维度应该对应数据的不同独立属性（如人脸图像中，一个维度控制微笑程度，另一个维度控制头部角度）。

<details>
<summary><b>📊 自编码器的应用：作为生成模型？</b></summary>

训练好的解码器 $D(Z)$ 理论上是一个生成器：输入任意一个潜在编码 $Z$，它应该输出一个看起来像训练数据的样本。然而，我们不知道应该输入什么样的 $Z$。训练集中的每个点 $X$ 对应一个 $Z = E(X)$。但如果我们随机采样 $Z \sim N(0, I)$，输入解码器，它输出的图像通常是**无意义**的。这是因为我们从未约束过 $Z$ 的分布。
</details>

# 二、变分自编码器 (VAE)

## 2.1 核心思想：让潜在空间"规整"起来

VAE 的核心创新在于：它不仅学习编码和解码，还强制要求编码器输出的潜在编码 $Z$ 服从一个**先验分布**，通常是标准正态分布 $N(0, I)$。这使得潜在空间变得连续且规整，从而可以通过从 $N(0, I)$ 中采样来生成新样本。

## 2.2 VAE 的生成故事

1.  从一个**先验分布** $p(Z) = N(0, I)$ 中采样一个潜在编码 $Z$。
2.  通过**生成器 (解码器)** $p_\theta(X|Z)$ 产生样本。通常假设 $p_\theta(X|Z)$ 是一个各向同性的高斯分布，其均值由解码器网络决定。
3.  我们观察到的数据 $X$ 就是这个过程的产物，并带有一些噪声。

## 2.3 变分下界 (ELBO) 与训练损失

由于我们无法直接计算后验 $p_\theta(Z|X)$，VAE 引入了一个**编码器网络 $q_\phi(Z|X)$** 来近似它。训练目标是最大化数据的对数似然的**证据下界 (ELBO)**，这等价于最小化以下损失函数：

$$
L_{\text{VAE}}(\theta, \phi; X) = \underbrace{-\mathbb{E}_{Z \sim q_\phi(Z|X)}[\log p_\theta(X|Z)]}_{\text{重构损失 (Reconstruction Loss)}} + \underbrace{\text{KL}(q_\phi(Z|X) \parallel p(Z))}_{\text{正则化项 (KL Divergence)}}
$$

1.  **重构损失 (Negative Log-Likelihood)**：衡量解码器 $p_\theta(X|Z)$ 重构输入的能力。对于图像，通常使用 MSE（假设高斯输出）或交叉熵（假设伯努利输出）。
2.  **KL散度**：衡量近似的后验分布 $q_\phi(Z|X)$ 和先验分布 $p(Z)$ (标准正态分布) 之间的差异。这是一个**正则化项**，迫使潜在空间 $Z$ 变得规整。

## 2.4 参数化与重参数化技巧

-   **编码器输出**：编码器不直接输出一个编码 $Z$，而是输出两个向量：均值 $\mu(X)$ 和对数方差 $\log\sigma^2(X)$。这意味着 $q_\phi(Z|X) = N(\mu, \sigma^2 I)$。
-   **KL散度公式**：对于两个各向同性高斯分布 $N(\mu, \sigma^2 I)$ 和 $N(0, I)$，KL散度有闭式解：
    <br>
    $$ \text{KL}(N(\mu, \sigma^2 I) \parallel N(0, I)) = -\frac{1}{2} \sum_{i=1}^{d} \left(1 + \log(\sigma_i^2) - \mu_i^2 - \sigma_i^2\right) $$
    <br>
-   **重参数化技巧 (Reparameterization Trick)**：为了通过随机采样 $Z \sim q_\phi(Z|X)$ 进行反向传播，我们将 $Z$ 写为：
    <br>
    $$ Z = \mu + \sigma \odot \epsilon, \quad \epsilon \sim N(0, I) $$
    <br>
    梯度现在可以通过 $\mu$ 和 $\sigma$ 流向编码器网络，而采样操作 $\epsilon$ 是固定的。

    ```
        VAE 网络结构：

        输入 X --> [编码器网络] --> μ(X)  -----> 采样 Z = μ + σ ⊙ ε --> [解码器网络] --> 重构 X_hat
                                   |                     ^
                                   +--> log(σ²(X)) --------+
                                                           |
                                            ε ~ N(0, I) (固定随机数)

        * 损失 = MSE(X, X_hat) + KL_term
    ```

## 2.5 VAE 的问题：模糊生成

-   **原因**：VAE 的损失函数在像素级别进行优化。为了最小化 MSE，模型倾向于生成所有可能输出的**平均**，这自然会导致图像模糊。
-   **对比**：VAE 优化的是"似然"，而 GAN 优化的是"真实性"。

# 三、扩散模型 (Diffusion Models)

## 3.1 灵感来源：非平衡热力学

扩散模型通过一个多步的**去噪**过程来生成数据。核心思想是：将一个复杂的生成问题（一步从噪声变图像）拆解成成千上万个简单的去噪步骤。

-   **前向过程 (扩散过程)**：是一个固定的、无参数的马尔可夫链。它逐步向真实图像 $x_0$ 中添加高斯噪声，直到图像 $x_T$ 变成纯噪声。
    <br>
    $$ q(x_t \mid x_{t-1}) = N(x_t; \sqrt{1-\beta_t} x_{t-1}, \beta_t I) $$
    <br>
-   **反向过程 (去噪过程)**：是一个可学习的马尔可夫链。模型学习从纯噪声 $x_T$ 开始，逐步去除噪声，最终还原出图像 $x_0$。
    <br>
    $$ p_\theta(x_{t-1} \mid x_t) = N(x_{t-1}; \mu_\theta(x_t, t), \Sigma_\theta(x_t, t)) $$
    <br>

    ```
        扩散模型的前向与反向过程：

        前向过程 (固定，加噪)：
        x0 (真实) --> x1 --> x2 --> ... --> xT (纯噪声)
          ^                                    |
          |                                    |
          |                                    v
        反向过程 (学习，去噪)：
        x0 (生成) <-- x1 <-- x2 <-- ... <-- xT (从噪声开始)
        pθ(x0:T)
    ```

## 3.2 训练目标：噪声预测

训练一个扩散模型等价于训练一个**噪声预测器** $\epsilon_\theta(x_t, t)$。给定一个加噪后的图像 $x_t$ 和时间步 $t$，模型需要预测出被添加的噪声 $\epsilon$。

-   **重参数化**：由于前向过程是线性的，我们可以直接得到从 $x_0$ 到 $x_t$ 的公式：
    <br>
    $$ x_t = \sqrt{\bar{\alpha}_t} x_0 + \sqrt{1 - \bar{\alpha}_t} \epsilon, \quad \epsilon \sim N(0, I) $$
    <br>
    其中 $\alpha_t = 1 - \beta_t$，$\bar{\alpha}_t = \prod_{s=1}^t \alpha_s$。
-   **训练损失 (DDPM)**：在随机的 $t$ 下，从数据集中采样 $x_0$，采样噪声 $\epsilon$，计算 $x_t$，然后最小化预测噪声与真实噪声的 MSE。
    <br>
    $$ L_{\text{simple}} = \mathbb{E}_{t, x_0, \epsilon} \left[ \| \epsilon - \epsilon_\theta(\sqrt{\bar{\alpha}_t}x_0 + \sqrt{1-\bar{\alpha}_t}\epsilon, t) \|^2 \right] $$
    <br>

<details>
<summary><b>💡 直觉理解：噪声预测与得分匹配 (Score Matching)</b></summary>

-   优化模型 $\epsilon_\theta(x_t, t)$ 预测噪声，等价于让模型学习**得分函数 (Score Function)** $\nabla_{x_t} \log p(x_t)$，即对数概率密度在数据空间中的梯度。
-   **物理意义**：得分函数 $\nabla \log p(x)$ 指向数据分布中概率密度**增加最快**的方向。因此，沿着得分函数的反方向移动，就能增加样本的概率，即**去噪**。
-   这种视角将扩散模型与朗之万动力学 (Langevin Dynamics) 联系起来，为生成过程提供了坚实的数学基础。
</details>

## 3.3 加速采样：DDIM

-   **问题**：标准的 DDPM 采样需要迭代 1000 步，非常慢。
-   **解决方案 (DDIM)**：将反向过程重新定义为**非马尔可夫链**，允许我们**跳过**中间的许多时间步。
    <br>
    $$ x_{t-1} = \sqrt{\bar{\alpha}_{t-1}} \left( \frac{x_t - \sqrt{1-\bar{\alpha}_t} \epsilon_\theta(x_t, t)}{\sqrt{\bar{\alpha}_t}} \right) + \sqrt{1 - \bar{\alpha}_{t-1} - \sigma_t^2} \cdot \epsilon_\theta(x_t, t) + \sigma_t \epsilon $$
    <br>
    当 $\sigma_t = 0$ 时，采样过程变为**确定性**的，且可以用 10-50 步就生成高质量图像。

# 四、生成对抗网络 (GAN)

## 4.1 核心思想：对抗游戏

GAN 由一个**生成器 (Generator, G)** 和一个**判别器 (Discriminator, D)** 组成，二者进行博弈。

-   **生成器 (G)**：接收随机噪声 $z$，目标是生成以假乱真的图像，从而**欺骗**判别器。
-   **判别器 (D)**：接收真实图像和生成图像，目标是学会**区分**二者。
-   **纳什均衡**：理想情况下，最终生成器能生成完美图像，判别器的准确率只能达到 50%（随机猜测）。

## 4.2 目标函数与训练

-   **Min-Max 博弈**：
    <br>
    $$ \min_G \max_D V(D, G) = \mathbb{E}_{x \sim p_{\text{data}}(x)}[\log D(x)] + \mathbb{E}_{z \sim p_z(z)}[\log(1 - D(G(z)))] $$
    <br>
-   **训练步骤** (每次迭代):
    1.  **训练判别器**：固定生成器，用一个 batch 的真实图像和一个 batch 的生成图像训练 D，使其最大化上述目标函数。
    2.  **训练生成器**：固定判别器，训练 G 使其最小化 $\log(1 - D(G(z)))$，即"欺骗" D。

## 4.3 理论分析：最优判别器与 JS 散度

-   **最优判别器**：对于任意给定的生成器 $G$，最优的判别器 $D_G^*$ 为：
    <br>
    $$ D_G^*(x) = \frac{p_{\text{data}}(x)}{p_{\text{data}}(x) + p_g(x)} $$
    <br>
-   **收敛性**：当 $p_g = p_{\text{data}}$ 时，$D_G^*(x) = 0.5$。
-   **损失的重新理解**：将 $D_G^*$ 代入目标函数，可以得到：
    <br>
    $$ C(G) = \max_D V(D, G) = 2 \cdot \text{JS}(p_{\text{data}} \parallel p_g) - \log 4 $$
    <br>
    其中 **JS 散度 (Jensen-Shannon Divergence)** 是对称的，能更好地衡量两个分布的距离。

## 4.4 经典 GAN 的问题

1.  **模式坍塌 (Mode Collapse)**：生成器只学会产生少数几种甚至一种模式（如总是生成同一张脸），从而轻易骗过判别器，但丧失了多样性。
2.  **梯度消失 (Vanishing Gradients)**：当判别器太强，其输出的 $D(G(z))$ 总是接近 0，导致生成器的梯度 $\nabla_G \log(1-D(G(z)))$ 趋于 0，生成器无法学习。

    ```
        模式坍塌示意：

        真实数据分布 (两个模式):  ●●  ●●
                              ●●  ●●

        健康的生成分布:       ●  ●●  ●
                              ●  ●  ●

        模式坍塌的生成分布:    ●   ●
                              ●   ●   (只捕获了一个模式)
    ```

## 4.5 GAN 的改进：WGAN

Wasserstein GAN 通过使用 Wasserstein 距离 (Earth Mover's Distance) 替代 JS 散度，从根本上解决了梯度消失问题。

-   **Wasserstein 距离**：衡量将一个概率分布"搬运"成另一个分布所需的最小代价。即使两个分布完全不相交，它也能提供一个有意义的、平滑的距离度量。
-   **WGAN 的改动**：
    1.  移除判别器的 Sigmoid 激活函数，将其改为一个**评论家 (Critic)**，输出一个实数评分（代表"真实性"）。
    2.  使用 Wasserstein 损失的近似形式。
    3.  强制评论家函数是 **1-Lipschitz 连续**的（通过权重裁剪或梯度惩罚 WGAN-GP）。
-   **效果**：训练更稳定，生成质量更高，有效缓解模式坍塌。

# 五、生成模型对比总结

| 模型          | 生成质量      | 多样性           | 训练稳定性     | 推断速度              | 似然估计   | 潜在空间                 | 核心挑战             |
| :------------ | :------------ | :--------------- | :------------- | :-------------------- | :--------- | :----------------------- | :------------------- |
| **VAE**       | 一般 (模糊)   | 高               | 稳定           | 快 (单步)             | 可以计算   | 连续、结构好             | 生成图像模糊         |
| **Diffusion** | 非常高 (SOTA) | 高               | 较稳定         | 慢 (多步，DDIM可加速) | 可以计算   | 无明确含义 (与 VAE 不同) | 采样速度慢           |
| **GAN**       | 非常高 (清晰) | 可能差(模式坍塌) | 不稳定，难训练 | 快 (单步)             | 不可以计算 | 连续、纠缠               | 训练不稳定，模式坍塌 |

### 生成模型的演变逻辑

1.  **AE -> VAE**：为了能生成，必须让潜在空间规整，因此引入 KL 散度。
2.  **VAE -> Diffusion**：VAE 的一步生成太"贪心"，导致模糊。Diffusion 将生成过程分解为 1000 个小步，每一步只学加一点点噪声，大大简化了学习任务。
3.  **VAE -> GAN**：VAE 的像素级损失不是衡量"真实性"的好指标。GAN 让一个网络去学"什么看起来真实"，然后用这个网络作为损失函数，能产生更清晰的图像。
